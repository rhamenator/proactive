# Security Policy

## Supported Versions

The latest release on the default branch is supported for security updates.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately via GitHub Security Advisories:

1. Go to the repository Security tab.
2. Select Report a vulnerability.
3. Provide reproduction steps, impact, and any suggested mitigations.

If private reporting is unavailable, open a private channel with the maintainers and avoid posting exploit details in public issues.

## Response Process

- We will acknowledge receipt as soon as possible.
- We will validate and triage severity.
- We will coordinate remediation and disclosure timelines.
